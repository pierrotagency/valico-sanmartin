# valico-sanmartin

> valico extension

## Install

```bash
npm install --save valico-sanmartin
```

## Usage

```jsx
import React, { Component } from 'react'

import MyComponent from 'valico-sanmartin'

class Example extends Component {
  render () {
    return (
      <MyComponent />
    )
  }
}
```
