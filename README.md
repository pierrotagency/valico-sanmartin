# valico-sanmartin

## Run local

```bash
yarn install 
```


```bash
yarn start 
```

## Usage

```jsx
import React, { Component } from 'react'

import MyComponent from 'valico-sanmartin'

class Example extends Component {
  render () {
    return (
      <MyComponent />
    )
  }
}
```
